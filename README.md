# TinyML Systolic Array Accelerator

## Project Title

Design and ASIC Implementation of a Configurable Low-Power TinyML Systolic Array Accelerator Using Open-Source RTL-to-GDSII Flow

## Objective

This project aims to design, verify, and implement a configurable low-power TinyML systolic array accelerator targeting edge AI applications using an entirely open-source RTL-to-GDSII flow.
